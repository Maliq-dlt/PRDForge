# Security Policy

## Supported Versions

Only the latest release version receives security updates.

| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅ Yes    |
| < 1.0.0 | ❌ No     |

## Reporting a Vulnerability

We take the security of AI PRD Builder seriously. If you find a security vulnerability, please do NOT open a public issue. Instead, report it privately:

1. Send an email to **security@example.com** with details of the vulnerability.
2. Include a description of the vulnerability, steps to reproduce, and potential impact.
3. We will acknowledge your report within 48 hours and work on a fix.
4. We aim to publish a patched release within 14 days of receiving a verified report.
